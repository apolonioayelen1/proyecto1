# Mis Avances en SQL - Base de Datos AdventureWorks

¡Hola! En este repositorio comparto mis soluciones a los ejercicios de consultas en bases de datos utilizando SQL Server.

## 🛠️ Tecnologías Utilizadas
* **Motor de Base de Datos:** SQL Server (SSMS)
* **Control de Versiones:** Git & GitHub
* **Editor de Código:** Visual Studio Code

## 📁 Estructura del Proyecto
* `consultas_adventureworks.sql`: Archivo principal con las consultas estructuradas y corregidas.
* `.gitignore`: Configuración para mantener el entorno limpio de archivos basura.
